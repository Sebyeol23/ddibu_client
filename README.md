# DDIBU

> 박근영

> REACT, Express, MariaDB

## 홈
> 다양한 페이지로 이동 가능하며 등록된 상품을 확인할 수 있고 로그인 및 회원가입이 가능합니다.

- ### 홈 화면

  <img src="https://github.com/Sebyeol23/ddibu_client/assets/98662998/07c2a52e-ff88-4426-b1f2-033791a1c4ba" width="600" height="400">

   - Nav Bar의 상품 등록, 좋아요, 채팅, 마이페이지 클릭시 해당 화면으로 이동 가능
   - Main Section에 등록된 상품 나열
   - 우측 상단의 회원가입 및 로그인 버튼 클릭 가능

- ### 회원가입

  <img src="https://github.com/Sebyeol23/ddibu_client/assets/98662998/bc404926-0817-42b2-a0bf-78f24866a39e" width="600" height="400">

  - 아이디와 비밀번호 입력 가능
  - 회원가입 버튼을 눌러 회원가입이 완료되면 로그인 페이지로 이동

- ### 로그인
  
  <img src="https://github.com/Sebyeol23/ddibu_client/assets/98662998/01eb2fe7-4705-461c-9acf-eb91cac7f793" width="600" height="400">

  - 마찬가지로 아이디와 비밀번호 입력 가능
  - 로그인 완료시 홈 화면으로 이동

- ### 로그인 상태
  
  <img src="https://github.com/Sebyeol23/ddibu_client/assets/98662998/5935f1a3-5541-466f-b721-1a63f3b6cde3" width="600" height="400">

  - 회원가입, 로그인 버튼이 유저 아이디와 로그아웃 버튼으로 변경
  - 로그아웃 버튼을 눌러 로그아웃 가능

- ### 로그아웃 상태
  
  <img src="https://github.com/Sebyeol23/ddibu_client/assets/98662998/ef967333-2851-4486-a2ed-f1c87ae58e1e" width="600" height="400">

  - 유저 아이디와 로그아웃 버튼이 회원가입과 로그인 버튼으로 변경
  - 일부 기능 이용 불가

## 상품 등록
> 홈 화면의 상품 등록을 클릭하면 이동하는 페이지입니다.
> 등록하고 싶은 상품의 정보를 입력하여 상품을 등록할 수 있습니다.

- ### 상품 등록

  <img src="https://github.com/Sebyeol23/ddibu_client/assets/98662998/2940a3ca-a893-41cc-b6f5-f734017e2b63" width="600" height="400">

   - 상품의 이미지, 이름, 설명, 가격, 태그 입력 가능
   - 태그 추가를 누를 경우 태그 입력 칸 생성
   - 여러개의 태그 입력 가능

- ### 상품 정보 입력

  <img src="https://github.com/Sebyeol23/ddibu_client/assets/98662998/aa0610d4-fccf-4b68-b9ec-b773a349400e" width="600" height="400">

  - 상품 등록 버튼을 눌러 상품 등록 가능

- ### 상품 등록 결과
  
  <img src="https://github.com/Sebyeol23/ddibu_client/assets/98662998/19664f2e-30f8-46c5-9d90-8c9b37adab79" width="600" height="400">

  - 홈 화면에 등록한 상품이 나열됨

## 좋아요
> 홈 화면의 좋아요를 클릭하면 이동하는 페이지입니다.
> 좋아요를 누른 상품 목록을 보여주며 홈 화면과 같이 상품을 클릭하면 상품 정보 페이지로 이동합니다.

- ### 좋아요
  <img src = "https://github.com/Sebyeol23/ddibu_client/assets/98662998/3866da7c-15c2-4385-947b-3ff797f9aff9" width = "600" height = "400" >

## 채팅

> 홈 화면의 채팅을 클릭하면 이동하는 페이지입니다.
> 참여중인 채팅방 목록을 보여주며 특정 채팅방을 클릭해 채팅 기록을 확인할 수 있습니다.

<br />

- ### 채팅방 리스트
  <img src = "https://github.com/Sebyeol23/ddibu_client/assets/98662998/0582b8ac-e4df-4900-a0b5-196c5e053da1" width = "600" height = "400">

  - [상품아이디] 대화 상대 아이디 형태로 채팅방이 표시됨
  - 특정 채팅방 클릭시 채팅방으로 이동

- ### 채팅방
  <img src = "https://github.com/Sebyeol23/ddibu_client/assets/98662998/d3e478eb-6872-4dfe-bcff-144a87cc054f" width = "600" height = "400">

  - 채팅 기록 확인 가능
  - 실시간 채팅 가능
